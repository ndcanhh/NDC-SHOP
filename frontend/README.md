# NDC SHOP - Frontend 📱

Thư mục này chứa toàn bộ mã nguồn giao diện người dùng (Frontend) của hệ thống NDC SHOP.

## 🚀 Công nghệ sử dụng
- **React.js (Vite)**: Framework chính.
- **React Bootstrap**: Styling và Layout.
- **Context API**: Quản lý giỏ hàng và xác thực người dùng.
- **React Router Dom**: Điều hướng trang.
- **Axios**: Kết nối API Backend.
- **Cloudinary**: Tối ưu hóa hiển thị hình ảnh.

## 📦 Cài đặt & Chạy Local
1. Cài đặt dependencies: `npm install`
2. Chạy môi trường phát triển: `npm run dev`
3. Xây dựng bản production: `npm run build`

## 🛠️ Cấu hình
Các cấu hình Proxy và biến môi trường được thiết lập trong `vite.config.js`.
Khi deploy lên Vercel, các thiết lập chuyển hướng (rewrites) được quản lý bởi `vercel.json`.
